# ReviewBot - AI 리뷰 응답 자동화 SaaS 디자인 브리프

## 프로젝트 개요

앱 스토어 리뷰에 AI가 자동으로 응답을 생성해주는 **B2B SaaS 프로덕트**.
고객사의 Google Play / App Store 리뷰를 수집하고, 브랜드 톤에 맞는 응답을 자동 생성하여 운영 비용을 줄여주는 서비스.

## 플랜 구조

### Lite (라이트)
> 내친소 같은 소규모 앱 대상. 빠르고 간단한 리뷰 응답.

- 긍정 리뷰(4~5점)에 감사 응답 자동 생성
- 고정 페르소나 이름만 설정 (이름 등록만, 성격 커스터마이징 없음)
- 짧은 리뷰 → 고정 응답 풀에서 랜덤 선택 (LLM 호출 없음)
- 비꼬는 리뷰 자동 감지 및 스킵
- 기본 대시보드 (처리량, 쿼터)
- 카테고리 자동 분류 (기본 9개 고정)

### Premium (프리미엄)
> 머니워크 같은 전문 운영 앱 대상. 페르소나 + RAG 기반 정밀 응답.

Lite의 모든 기능 포함, 추가로:

- **페르소나 상세 설정**
  - 성격/톤 유형 선택 (친근하고 공감형 / 정중하고 전문적인 / 간결하고 명확한 / 활기차고 긍정적)
  - 응답 가이드라인 상세 편집 (배려 기본 감정어, 금지어/표현)
  - 핵심 FYI 메모 (페르소나에게 항상 전달할 맥락 정보)
  - 실시간 미리보기 (설정 변경 시 바로 샘플 응답 확인)
- **RAG 지식베이스 연동**
  - 고객사 FAQ, 공식 문서, 공지사항 등 업로드
  - 리뷰 내용에 맞는 관련 지식을 검색하여 응답에 반영
  - 정확한 정보 기반 응답 (단순 감사가 아닌 구체적 안내 가능)
- **카테고리 커스터마이징**
  - 기본 카테고리 외 고객사 맞춤 카테고리 추가/편집
  - 카테고리별 응답 전략 다르게 설정
- **전체 별점 대응 (1~5점)**
  - 부정 리뷰(1~3점)에도 대응 가능
  - 카테고리별 맞춤 응답 (기능 오류 → 사과 + 해결 안내, 문의 → FAQ 연결 등)
- **고급 대시보드**
  - 카테고리별 분포 분석
  - 응답 비용 분석 (LLM 호출 vs 고정)
  - RAG 히트율 (지식베이스 활용 비율)
  - 감정 분석 트렌드

| **워크스페이스 오너** | 고객사 담당자. 앱 등록, API 연동, 페르소나/톤 설정 |

## 핵심 기능 상세

### 1. 워크스페이스 (고객사 관리)
- 고객사별 독립 워크스페이스 (예: "내친소", "머니워크" 등)
- 워크스페이스별 앱 등록 (Google Play 패키지명, App Store 번들ID)
- API 키 연동 (Google Play Service Account, App Store Connect Key)
- 멤버 초대 및 권한 관리
- 플랜 선택 (Lite / Premium)

### 2. 페르소나 설정 (Premium)
참고: 이미 디자인된 화면 있음 (페르소나 설정 페이지)
- 성격/톤 유형 4가지 중 선택
  - 친근하고 공감형
  - 정중하고 전문적인
  - 간결하고 명확한
  - 활기차고 긍정적
- 응답 가이드라인 상세 설정
  - 배려/기본 감정어
  - 금지어/표현 피드백
- 핵심 FYI: 페르소나에게 항상 전달할 맥락 (예: "현재 v2.0 업데이트 진행 중")
- 오른쪽 패널: 실시간 미리보기

### 3. RAG 지식베이스 (Premium)
- 문서 업로드: FAQ, 공지사항, 이용약관, 업데이트 노트 등
- URL 크롤링: 고객사 헬프센터 URL 등록하면 자동 수집
- 벡터 검색: 리뷰 내용과 관련된 지식을 자동으로 찾아 응답에 반영
- 지식 관리: 업로드된 문서 목록, 마지막 업데이트 날짜, 삭제

### 4. 리뷰 목록 (메인 화면)
- Google Play / App Store 탭 전환
- 리뷰 카드: 별점, 작성자, 리뷰 내용, 작성일
- 필터: 별점, 답변 상태(미답변/답변완료/스킵), 날짜 범위, 카테고리
- 정렬: 최근순 (기본)
- 리뷰 유형 뱃지: "짧은 리뷰", "비꼬는 리뷰(스킵)" 자동 태깅

### 5. AI 응답 생성 및 관리
- 리뷰 옆에 AI 생성 응답 미리보기
- 배정된 페르소나 표시
- Premium: 응답에 사용된 RAG 소스 표시 (어떤 문서를 참조했는지)
- 응답 편집 가능 (게시 전 수정)
- 일괄 생성 / 일괄 게시
- 재생성 버튼

### 6. 대시보드/통계
**Lite:**
- 오늘 처리량 / 남은 API 쿼터
- 플랫폼별 응답 현황
- 별점 분포 차트
- 최근 7일간 리뷰 추이

**Premium 추가:**
- 카테고리별 분포 분석
- 응답 비용 분석 (LLM vs 고정)
- RAG 히트율
- 감정 분석 트렌드
- 부정 리뷰 대응율

## 데이터 모델

### Workspace (워크스페이스)
| 필드 | 타입 | 설명 |
|------|------|------|

### Persona (페르소나)
| 필드 | 타입 | 설명 |
|------|------|------|
| name | string | 페르소나 이름 |
| tone_type | string | 친근공감 / 정중전문 / 간결명확 / 활기긍정 (Premium) |
| guidelines | string | 응답 가이드라인 (Premium) |
| forbidden_words | string[] | 금지 표현 (Premium) |
| fyi_notes | string | 핵심 FYI 메모 (Premium) |

### KnowledgeBase (지식베이스) - Premium only
| 필드 | 타입 | 설명 |
|------|------|------|
| workspace_id | string | 소속 워크스페이스 |
| documents | Document[] | 업로드된 문서 목록 |
| urls | string[] | 크롤링 대상 URL |
| last_updated | datetime | 마지막 업데이트 |

### Review (리뷰)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 리뷰 고유 ID |
| workspace_id | string | 소속 워크스페이스 |
| author | string | 작성자 |
| rating | int | 별점 (1~5) |
| content | string | 리뷰 내용 |
| created_at | datetime | 작성일 |
| platform | string | google_play / app_store |
| category | string | 자동 분류 카테고리 |
| status | string | pending / responded / skipped |

### ReviewResponse (응답)
| 필드 | 타입 | 설명 |
|------|------|------|
| review_id | string | 연결된 리뷰 ID |

| type | string | ai_generated / rag_enhanced / fixed_template |
| rag_sources | string[] | 참조된 문서 (Premium) |
| status | string | draft / approved / posted |

## 화면 구성

### 주요 페이지
- **온보딩**: 워크스페이스 생성 → 플랜 선택 → 앱 등록 → API 연동 → 페르소나 설정
- **리뷰 관리**: 메인 작업 화면
- **대시보드**: 통계 및 차트
- **페르소나 설정**: Lite는 이름만 / Premium은 상세 설정 (디자인 완료)
- **지식베이스 관리**: Premium only. 문서 업로드, URL 크롤링
- **멤버 관리**: 초대, 권한
- **요금제/빌링**: 플랜 선택, 사용량

### 전체 레이아웃
1. **사이드바**: 워크스페이스 선택, 메뉴
2. **상단 바**: 플랫폼 탭, 필터, 검색, 일괄 작업 버튼
3. **메인 영역**: 리뷰 목록 + 응답 미리보기
4. **통계 카드**: 오늘 처리량, 쿼터 잔량, 스킵 수

## 기술 스택

- **백엔드**: Python (LangChain, GPT-4o-mini)
- **프론트엔드**: Next.js
- **벡터DB**: RAG용 벡터 저장소 (Premium)
- **API 연동**: Google Play Developer API, App Store Connect API

## 디자인 방향

- SaaS 대시보드 스타일, 깔끔하고 프로페셔널한 톤
- Lite / Premium 구분이 UI에서 자연스럽게 드러남 (Premium 기능은 잠금 아이콘 또는 업그레이드 유도)
- 중립적 컬러 베이스 + 파란색 계열 액센트 (기존 디자인 참조)
- 상태 뱃지: 미답변(주황), 답변완료(초록), 스킵(회색), 초안(파랑)
- 다크모드 지원 고려

## 기존 디자인 참고

- 페르소나 설정 페이지: 이미 Stitch에서 디자인 완료 (Premium 버전)
  - 왼쪽: 성격 유형 선택, 가이드라인, FYI, 금지어
  - 오른쪽: 실시간 미리보기
  - 하단: 취소 / 페르소나 저장 버튼
